# 参与贡献

非常欢迎你来一起改进这个小工具！无论是修 bug、加功能还是改文档，都很受欢迎。

## 怎么提建议或报告问题

1. 打开本仓库的 **Issues** 页面
2. 点 **New Issue**，描述清楚你遇到的问题或想法
3. 如果是 bug，最好附上：你运行的命令、报错信息、你的操作系统

## 怎么提交代码（Pull Request）

1. Fork 本仓库到你自己的账号
2. 新建一个分支：`git checkout -b my-feature`
3. 修改代码，本地测试通过
4. 提交：`git commit -m "简短说明你改了什么"`
5. 推送：`git push origin my-feature`
6. 在 GitHub 上发起 Pull Request

## 一些约定

- 代码保持简单易读，多写中文注释，方便初学者看懂
- 新增文件类型时，请加到 `organizer.py` 的 `DEFAULT_RULES` 里
- 改动后请至少手动跑一遍 `--dry-run` 确认没问题

谢谢你的参与！🎉
